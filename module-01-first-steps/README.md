# Module 1 · First Steps

> **You'll learn:** why frameworks like Vue exist, how to create a real Vue project on your machine, and how to make a page that reacts to data - the "aha" moment the whole course builds on.

## Lessons

1. [Why Vue?](./01-why-vue.md) - the problem Vue solves, and Vue's mental model in one diagram *(~10 min, no install needed)*
2. [Your First Project](./02-your-first-project.md) - scaffold a project, tour the files, see live reload *(~15 min)*
3. [Templates & Reactivity](./03-templates-and-reactivity.md) - display data, react to clicks, render lists *(~15 min + exercise)*

Take them in order - each one leans on the previous.

## When you're done

You'll have a running Vue project on your machine and a page you built yourself that updates instantly when data changes. That's the foundation for everything else.
