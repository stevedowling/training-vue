# Module 2 · Reactivity

> 📝 **Not written yet** - when you get here, ask Claude to *"write module 2 of the Vue course"*.

**Builds on:** [Module 1 · First Steps](../module-01-first-steps/)

## What you'll learn

- Derive values that update themselves with `computed` (e.g. a cart total that recalculates when items change)
- Choose between `ref` and `reactive`, and know why the course leads with `ref`
- Run side effects when data changes with `watch` (e.g. save to localStorage on every change)
- Debug the classic reactivity gotchas: missing `.value`, replaced objects, mutated props

## Planned lessons

1. Computed properties - values that derive themselves
2. `ref` vs `reactive` - and why `.value` exists at all
3. Watchers - reacting with side effects
