# Module 3 · Components

> 📝 **Not written yet** - when you get here, ask Claude to *"write module 3 of the Vue course"*.

**Builds on:** [Module 2 · Reactivity](../module-02-reactivity/)

## What you'll learn

- Split a growing page into components, and develop a feel for *where* to split
- Pass data down into a component with **props**
- Send events back up with **emits** - the "props down, events up" pattern that structures every Vue app
- Leave holes in a component for callers to fill with **slots**

## Planned lessons

1. Your first component - extracting and reusing
2. Props - passing data down
3. Emits - sending events up
4. Slots - flexible components
