# Module 6 · App-wide State

> 📝 **Not written yet** - when you get here, ask Claude to *"write module 6 of the Vue course"*.

## What you'll learn

- Recognise when "props down, events up" stops scaling (passing props through five layers = prop drilling)
- Create a **Pinia** store: shared state any component can read and update
- Put derived values and actions in the store, keeping components thin
- Know what *belongs* in a store vs. staying local to a component

## Planned lessons

1. The prop-drilling problem - why stores exist
2. Your first Pinia store
3. What goes in the store (and what doesn't)
