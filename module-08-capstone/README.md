# Module 8 · Capstone: Build & Deploy

> **You'll do:** plan, build, and publicly deploy one complete app - Snackmart, a real-API snack shop with cart, checkout, search, and detail pages - proving every module's skill without a scaffold to lean on.

**Builds on:** everything - [Forms](../module-04-forms-and-input/), [Routing](../module-05-routing/), [State](../module-06-app-wide-state/), [APIs](../module-07-talking-to-apis/)

## Lessons

1. [Planning the app](./01-planning-the-app.md) - routes, component tree, state homes, API plan; deliverable: PLAN.md *(~1 hour)*
2. [The build](./02-the-build.md) - six checkpoints with acceptance criteria; mostly you, hints at the seams *(a few sessions)*
3. [Build and deploy](./03-build-and-deploy.md) - npm run build, the SPA-fallback gotcha, and a URL you can text to people *(~1 hour)*

Take them strictly in order - the plan is lesson 2's input, and lesson 3 deploys lesson 2's output.

## When you're done

There's a URL. On it: a Vue app you planned on paper, built through six committed checkpoints, hardened against slow networks and bad URLs, and shipped with the same loop professionals use. Plus a SNAGS.md of your own hard-won debugging notes - the most personalised Vue reference in existence. Course complete. 🎓
