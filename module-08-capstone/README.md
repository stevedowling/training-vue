# Module 8 · Capstone: Build & Deploy

> 📝 **Not written yet** - when you get here, ask Claude to *"write module 8 of the Vue course"*. It's best written *after* modules 4–7, tailored to what you built along the way.

**Builds on:** everything - [Forms](../module-04-forms-and-input/), [Routing](../module-05-routing/), [State](../module-06-app-wide-state/), [APIs](../module-07-talking-to-apis/)

## What you'll do

Build one complete small app from scratch - multiple pages, a form, shared state, live API data - then put it on the internet:

- Plan the component tree before writing code
- Build it, hitting every module's skills at least once
- Run `npm run build` and understand what it produces
- Deploy the result to a free static host (GitHub Pages or Netlify) and share the URL

## Planned lessons

1. Planning the app - components, pages, state, data
2. The build (guided checkpoints, mostly you)
3. `npm run build` and deploying to the web
